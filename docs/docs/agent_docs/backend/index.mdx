## 🔍 Quick Reference

### **Common Commands**
```bash
# Development
make flask-app                   # Start backend
pytest tests/unit_tests/         # Run unit tests
ruff check .                     # Lint code
superset db upgrade              # Apply migrations

# Database
superset db migrate -m "message" # Create migration
superset init                    # Initialize permissions
superset load-examples           # Load sample data

# Testing
scripts/tests/run.sh             # Integration tests
pytest -k "test_name"            # Run specific test
pytest --cov=superset            # Coverage report
```

### **Key Patterns**
- **Command**: `CreateChartCommand(user, data).run()`
- **DAO**: `ChartDAO.find_by_id(chart_id)`
- **API**: `@expose('/endpoint') @protect()`
- **Security**: `security_manager.can_access_chart(chart)`
- **Cache**: `@cache.memoize(timeout=300)`

### **Important Files**
- `/superset/models/` - Database models
- `/superset/commands/` - Business logic
- `/superset/views/` - API endpoints
- `/superset/daos/` - Data access objects
- `/superset/db_engine_specs/` - Database connectors
